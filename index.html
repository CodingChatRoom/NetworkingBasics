Networking for Ethical Hacking

1. What is Networking?

Networking connects devices to share information and resources like files, internet, or printers. Devices communicate using protocols such as TCP/IP. It is essential for building secure systems and performing penetration testing.

  • Used in LANs, WANs, and the Internet
  • Supports communication, sharing, and access control
  • Helps ethical hackers understand data paths and vulnerabilities
Client 1 ----> [Switch] <---- Client 2 | [Router] --- Internet

2. IP Address and MAC Address

An IP Address identifies your device on a network. It can change over time or based on network location. A MAC Address is assigned to the network interface card (NIC) by the manufacturer. It's used within LAN communication.

  • IP Address Example: 192.168.1.10
  • MAC Address Example: 00:1A:2B:3C:4D:5E

Attackers often spoof IP or MAC addresses to hide identity or bypass access controls.

Device: Laptop IP: 192.168.1.10 MAC: 00:1A:2B:3C:4D:5E

3. Types of Networks

Networks are categorized based on their range and purpose:

  • PAN: Personal Area Network (Bluetooth, Tethering)
  • LAN: Local Area Network (Home, Office)
  • MAN: Metropolitan Area Network (Campus, City)
  • WAN: Wide Area Network (Internet)
PAN -> Mobile + Earphones LAN -> PCs in an Office MAN -> Network across University WAN -> Internet Worldwide

4. OSI Model

The OSI model helps visualize how communication happens over a network. It includes 7 layers:

  • 1. Physical
  • 2. Data Link
  • 3. Network
  • 4. Transport
  • 5. Session
  • 6. Presentation
  • 7. Application
+---------------------+ | 7. Application | | 6. Presentation | | 5. Session | | 4. Transport | | 3. Network | | 2. Data Link | | 1. Physical | +---------------------+

5. Protocols

Protocols are rules that define how data is transferred. Important ones include:

  • HTTP/HTTPS - Web browsing
  • FTP - File transfer
  • DNS - Domain name resolution
  • TCP/UDP - Data transport
  • ICMP - Ping requests

Hackers analyze or exploit protocols to intercept or manipulate data.

6. Hacking Tools

  • Nmap: Scans ports and services
  • Wireshark: Captures and analyzes packets
  • Netcat: Opens raw TCP/UDP connections
  • Aircrack-ng: Cracks Wi-Fi passwords
  • Burp Suite: Web app vulnerability testing

7. Importance of Networking for Hackers

Without networking knowledge, ethical hackers cannot perform tasks such as port scanning, sniffing, spoofing, or exploiting web apps. Networking is foundational to hacking.

8. Ethernet

Ethernet is a wired LAN technology that uses RJ45 cables. It is reliable and fast, used mostly in local networks. MAC addresses are used to identify devices.

Router | Switch --- PC1 |--- PC2 |--- Printer

9. Ports

Ports help devices identify specific services. Common ports:

  • 80 - HTTP
  • 443 - HTTPS
  • 22 - SSH
  • 21 - FTP
  • 53 - DNS

10. Routing

Routing decides the path for data. Routers use routing tables to forward packets to the right destination.

[PC] --> [Router] --> [Internet] --> [Server]

11. Communication Between Devices

Devices communicate by exchanging packets. These contain:

  • Source/Destination IP
  • Source/Destination MAC
  • Protocol
  • Port Numbers
  • Data Payload

Hackers capture and analyze packets to extract data or detect weaknesses.
